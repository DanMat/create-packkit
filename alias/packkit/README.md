# packkit

Short alias for **[create-packkit](https://www.npmjs.com/package/create-packkit)** — scaffold a modern npm package or CLI.

```sh
npx packkit                 # interactive
npx packkit ts-lib my-lib   # from a preset
```

Configure it visually at **[danmat.github.io/create-packkit](https://danmat.github.io/create-packkit/)**.

This package just re-runs `create-packkit`; see that package (and the [repo](https://github.com/DanMat/create-packkit)) for the full docs.
